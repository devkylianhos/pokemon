Welke winkels we bewaken

PocketPop houdt webshops in de gaten op Pokémon-restocks en prijsdalingen. We beginnen bewust met één winkel die we echt goed doen — bol.com via de officiële API — en breiden daarna uit. Geen loze beloftes: hieronder zie je precies wat live is en wat op de planning staat.

bol zelflos van marketplace

Voor bol gebruiken we de officiële bol Retailer API in plaats van de site te scrapen. Daardoor zien we niet alleen óf een product op voorraad is, maar ook of bol het zélf verkoopt (aanbieding met retailer-id 0 / Fulfilled By Bol) los van de marketplace-verkopers die op dezelfde EAN meeliften. Zodra bol zelf van uitverkocht naar op voorraad gaat, of een prijs zakt, verschijnt het meteen in je dashboard met een Bol Drop-melding en een directe koop-link. Filter op set of producttype en zet je belangrijkste items op "hot" voor snellere checks.
